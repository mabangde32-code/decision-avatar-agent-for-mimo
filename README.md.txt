# 角色化身决策助手（Multi-Agent Decision Advisor）

## 一、项目是什么？
这是一个给普通人做重大决策的AI工具，不用自己纠结，让4个专业AI角色帮你一起分析：
1. 谨慎财务顾问：算钱、算风险
2. 冒险创新者：看机会、看未来
3. 务实项目经理：看好不好落地
4. 善解人意的朋友：看你开不开心

## 二、解决什么痛点？
普通人做职业、买房、投资、搬家决策时：
- 容易被情绪影响
- 只看一面，看不到全局
- 不会分析风险和机会成本
这个工具用4个AI视角，帮你全面、理性做决定。

## 三、项目亮点（适配小米MiMo）
1. 多智能体（Agent）协作：4个AI同时分析，长链推理
2. 自动生成决策矩阵表格，清晰直观
3. 完全可以在小米MiMo上运行，支持长文本分析
4. 场景刚需：职业规划、购房、创业、生活选择

## 四、使用方法
1. 输入你的决策问题（比如：要不要转行？要不要买房？）
2. 4个AI角色分别给出分析
3. 自动生成对比矩阵报告
4. 直接参考报告做决策

## 五、计划使用小米MiMo
本项目计划基于MiMo大模型开发，利用其长文本理解和多角色对话能力，提升决策分析效果。
