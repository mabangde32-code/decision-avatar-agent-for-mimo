智能体 / Bot 成品说明（直接复制到你 GitHub 末尾）
本项目已在扣子 Coze 平台完成完整可运行智能体 Bot，实现多角色化身圆桌决策能力：
内置四大对立决策角色
明确结论输出和风险等级直观可视化的智能体，用于分析和解决存在多方立场的复杂问题。
1. 确保智能体包含至少4个具有明确对立立场的角色，角色间需存在实质性冲突，避免中立或模糊表述。
2. 智能体分析多方立场后，必须通过归纳法找出各立场的共识部分，并给出明确、清晰的最终结论，禁止使用"看你自己"等模糊表述。
3. 采用🟢（绿色，低风险/优势）、🟡（黄色，中等风险/优势）、🔴（红色，高风险/优势）三种颜色直观标记各立场的风险等级和占优情况，便于用户快速识别。智能体工作流程
用户输入任意人生决策问题 → 四个角色独立视角分析 → 自动汇总生成决策对比矩阵表格 → 输出中立综合建议。
技术价值
属于典型多 Agent 角色协同 + 长链推理应用，非常适配小米 MiMo 大模型长文本、多角色对话、逻辑归纳能力。
已完整可运行、可复用、可迁移至 MiMo 部署使用。
